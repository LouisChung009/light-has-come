# 字體方案預覽

## 方案 A: 雙字體組合 (推薦!)

**標題字體**: Ma Shan Zheng (馬善政楷書)
**內文字體**: Noto Sans TC

### 特色
- ✨ 標題有手寫童趣感
- 📖 內文清晰易讀
- 🎯 專業與活潑兼具

### 適合
- 想要活潑但不失專業感
- 標題吸睛,內文舒適

### 範例
```
光·來了 (Ma Shan Zheng - 手寫感)
玩出喜樂、美善的品格 (Ma Shan Zheng)

這裡是專屬童樂的園地，結合萌趣的遊戲... (Noto Sans TC - 清晰)
```

---

## 方案 B: 全站統一 (ZCOOL XiaoWei)

**全站字體**: ZCOOL XiaoWei (站酷小薇)

### 特色
- 🎨 筆畫圓潤可愛
- 💝 整體風格統一
- 😊 非常適合兒童

### 適合
- 想要整體可愛風格
- 追求一致性

### 範例
```
光·來了 (ZCOOL XiaoWei - 圓潤)
玩出喜樂、美善的品格 (ZCOOL XiaoWei - 圓潤)

這裡是專屬童樂的園地，結合萌趣的遊戲... (ZCOOL XiaoWei - 圓潤)
```

---

## 方案 C: 溫馨路線 (Noto Serif TC)

**全站字體**: Noto Serif TC

### 特色
- 📚 有襯線,像故事書
- ❤️ 溫暖親切感
- 🏛️ 傳統但不老氣

### 適合
- 想要溫馨故事感
- 強調傳統價值

### 範例
```
光·來了 (Noto Serif TC - 襯線)
玩出喜樂、美善的品格 (Noto Serif TC - 襯線)

這裡是專屬童樂的園地，結合萌趣的遊戲... (Noto Serif TC - 襯線)
```

## 如何預覽

我會建立三個版本的 layout.tsx,您可以切換查看:
1. `layout-option-a.tsx` - 方案 A
2. `layout-option-b.tsx` - 方案 B  
3. `layout-option-c.tsx` - 方案 C

只需要重新命名檔案為 `layout.tsx` 就能切換!
